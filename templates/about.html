About me

  • Interests: Ham radio, analog electronics, synthesizer design, computer architecture, and computer science (moreso just programming but I'll dabble in algorithms)
  • Favorite musical artists: Currently Femtanyl, Masayoshi Takanaka, and Machine Girl, but this does change
  • Favorite digital artist: @MintCashew
  • Favorite editor: Vim with minimal config (it's fun to make big macros)
  • Favorite OS: Windows 10 with a dev environment set up using WSL running Ubuntu and the i3 window manager
  • Favorite programming languages: Python for most quick things, C++ for embedded, and Rust and Uiua because they're fun (although I don't use these often)
  • Favorite drink: Americano coffee
  • Favorite video game: Counter Strike 2

Clubs and such that I'm in:

  • LASA Electronics Club - Founder and president (if you're a student at LASA, we are very casual and you should stop by sometime! Same with the Radio Club)
  • Science Olympiad: I compete in Circuit Lab, Remote Sensing, and Silicon Showdown (Texas trial event for computer architecture)

Thank you for stopping by my corner of the internet, I hope your day goes well today =).
